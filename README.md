# Counter App, Color Changer & BMI Calculator

A beginner-friendly frontend project built using HTML, CSS, and JavaScript. It demonstrates three fundamental JavaScript projects: Counter App, Color Changer, and BMI Calculator.

## Features

### Counter App
- Increase Count
- Decrease Count
- Reset Count

### Color Changer
- Random Background Colors
- Dynamic Color Name
- Automatic Text Color Change

### BMI Calculator
- Height & Weight Input
- BMI Calculation
- BMI Category Display
- Form Validation

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download the project.
2. Open in VS Code.
3. Install Live Server.
4. Run index.html using Live Server.
